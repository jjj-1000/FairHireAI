# FairHireAI

FairHireAI is an AI-powered hiring platform that helps companies streamline recruitment using smart candidate analysis and automation.

## Features
- AI-based candidate screening
- Resume analysis
- Frontend and Backend integration
- Fast and modern UI
- API integration with Groq AI

## Tech Stack
- React.js
- Node.js
- Express.js
- Vite
- Groq API

## Setup

1. Install dependencies

```bash
npm install
```

2. Create .env file

Add your GROQ API key

3. Run the project

```bash
npm run dev
```
